# Change Log

## [0.1.0] - 2026-05-13

### Added
- Initial release
- Status bar quick prompt buttons with one-click trigger
- Sidebar panel for prompt management with CRUD operations
- Two execution modes: Write to Input and Direct Execute
- Built-in shortcuts: New Chat Tab (smart chat), Close All Tabs
- Custom prompt configuration via VS Code settings (`copilotKit.prompts`)
- Status bar position configuration (`copilotKit.statusBarPosition`)
- Three display modes per button: icon only, text only, icon + text
- Right-click support to send prompt with selected code
- Supports Codicon library with search filtering
