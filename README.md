# Copilot Quick Prompts 🚀

为 VS Code Copilot 聊天添加快捷提示词按钮，一键发送预设提示词，避免重复输入。

## 功能

在侧边栏添加快捷提示面板，提供常用提示词按钮：

| 按钮 | 功能 |
|------|------|
| 🔍 代码审查 | 审查代码，找出问题 |
| 📖 解释代码 | 解释代码功能 |
| 🧪 编写测试 | 生成单元测试 |
| ⚡ 优化代码 | 优化性能和可读性 |
| 📝 添加注释 | 添加中文注释 |
| 🔄 重构建议 | 提供重构建议 |

**使用方式：**
- **左键点击**：发送预设提示词到 Copilot 聊天
- **右键点击**：附带当前选中的代码一起发送

## 安装

从 VS Code 扩展市场安装。

## 配置

> 暂不支持自定义提示词，后续版本将支持。
